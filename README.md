# Sorting Study Repository

Welcome to the **Sorting Study** repository! This project is dedicated to exploring, implementing, and visualizing various sorting algorithms to better understand their logic and performance.

## 🚀 Projects Included

### 1. QuickSort Visualizer
A real-time, interactive visualization of the QuickSort algorithm. It uses a bar chart to demonstrate the "divide and conquer" strategy, showing the pivot selection, partitioning, and recursion steps.

* **View Online:** [Live Demo on GitHub Pages](https://luizdiego.github.io/sorting-study/quick-sort/)
* **Project Details:** You can find the specific documentation, logic explanation, and source code in the subfolder README:
    * [Click here for the QuickSort Documentation](./quick-sort/README.md)
