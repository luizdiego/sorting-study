# QuickSort Visualizer

A simple, interactive web-based visualizer for the QuickSort algorithm. This project helps you understand how QuickSort works step-by-step by animating the sorting process of a bar chart in real-time.

## Features

* **Visual Animation:** Watch the QuickSort algorithm sort an array of 10 bars with a 1-second delay between operations.
* **Color-Coded Status:**
  * 🟨 **Yellow:** Current Pivot element.
  * 🟥 **Red:** Elements currently being compared.
  * 🟩 **Green:** Elements placed in their final sorted position.
* **Custom Range:** Generates random integer values between 1 and 10.
* **No Dependencies:** Built with pure HTML, CSS, and Vanilla JavaScript.

## How to Run

1. Clone this repository or download the `index.html` file.
2. Double-click the `index.html` file to open it in any modern web browser.
3. Click on **"Gerar Nova Lista"** (Generate New List) to generate a new random set of bars.
4. Click on **"Ordenar (Quicksort)"** (Sort) to watch the sorting animation!

## Algorithm Details

This project implements the **In-Place QuickSort** algorithm. It modifies the original array by swapping elements via their indices, which is the standard approach for visual animations and memory efficiency.
